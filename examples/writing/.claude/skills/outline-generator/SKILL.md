## Outline Generator

**When to use:** When you have an idea but need help organizing it into a structure you can write from.

**Instructions:** Ask the user for: (1) the type of content (article, essay, newsletter, report, listicle, etc.); (2) target length; (3) target audience; (4) the main argument, takeaway, or hook. If any of these are missing, ask before generating the outline. Then produce a specific, detailed outline with: a working headline, a lead description (1-2 sentences on how the piece opens and what it promises), section headers with one-sentence descriptions of each section, and a closing direction. For essays: include the thesis. For listicles: include the list items. For newsletters: include the subject line and call-to-action. The outline should be detailed enough that the writer could produce the piece from it without needing further clarification. Offer 2-3 structural options if the topic could be approached in genuinely different ways.

**Example prompts:**

- "Outline an 800-word article on [topic] for [audience]."
- "I want to write an essay arguing [claim] — help me structure it."
- "Give me 3 different structural approaches for a piece on [topic]."
