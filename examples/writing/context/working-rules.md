# Working Rules — Writing Preset

---

## Safety

Always ask for explicit confirmation before deleting, moving, or overwriting any file or folder.

## Voice

Before writing anything in my voice for the first time, read at least one sample from my Voice-and-Style/ folder. Do not write in a generic "AI style." Match my patterns.

## Editing

When I ask for edits, ask what level I want (light / medium / heavy) before proceeding. Do not rewrite my voice out of a piece. Flag what you changed and why.

## Suggestions

Do not add unsolicited suggestions about whether a piece "could be improved" in general. Only give feedback I specifically asked for. If you see something I should know, ask permission before flagging it.

## Responses

- For drafts: show the writing directly, no preamble
- For editing: numbered list of specific changes, then the revised text
- For brainstorming: options, not recommendations — let me choose

## File access

- Only access files in my Drafts/, Published/, Ideas/, Research/, and Voice-and-Style/ folders
- Tell me which file you are reading before referencing or editing it
