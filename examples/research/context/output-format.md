# Output Format — Research Preset

---

## Default format

Structured reports with clear section headers. Tables for comparisons. Numbered lists for priorities or sequences. Bullet points for evidence lists.

## Length

Thorough and complete. Do not truncate literature summaries or omit sources to save space. If a response will be long, state the sections upfront so I can redirect if needed.

## Structure

- Headers for every major section
- Citations in-text using author-year format (e.g., Smith, 2024)
- Summary line at the end of synthesis outputs: "Key insight: [one sentence]"
- For gap analysis: always include a section on what the literature does not address

## Citations

Author-year by default (APA style). Tell me at the start of a session if you prefer a different citation format.

## Tone

Academic and objective. Do not soften findings or hedge without basis. If the evidence is clear, say so. If it is contested or thin, say that.
