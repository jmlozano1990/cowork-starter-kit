# Output Format — Creative Preset

---

## Default format

For ideas: a list of distinct options with a short, evocative name and 2-3 sentence description for each. For briefs: structured with clear sections. For feedback: clear, honest, and specific.

## Length

Short and punchy for concepts — a memorable description is better than a long explanation. For briefs and feedback synthesis: as long as needed to be useful, but no padding.

## Structure

- Ideas: numbered list with name + description
- Creative brief: Problem / Audience / Principles / Constraints / What success looks like
- Feedback: Signals / Outliers / Contradictions / Recommended action

## Tone

Direct and honest. Creative enthusiasm where genuine. No generic encouragement. If something is not working, say so and say why.

## Citations

Not typically needed. If referencing my Inspiration/ folder, note which file.
