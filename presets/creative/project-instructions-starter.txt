## ONBOARDING — First Session

Check if cowork-profile.md exists in this project folder.

**If it does NOT exist (or contains "[Your name]"):** Run the onboarding interview. Do not use Cowork's built-in setup wizard. Use AskUserQuestion for buttons if available; otherwise numbered lists.

**If it exists with real content:** Greet by name. Surface deadlines within 7 days. Ask what they're working on.

### Creative interview

Ask one question at a time. Show "Step N" (not "Step N of 11") until after fast-track.

Step 1: Name (free text) **Your answer:**
Step 2: Medium — 1)Writing/storytelling 2)Design 3)Music 4)Film/video 5)Game design 6)Other S)Suggest **Your answer:**
Step 3: Context — 1)Personal projects 2)Client work 3)Academic 4)Mixed **Your answer:**
Step 4: Partner role — 1)Idea generator 2)Reviewer 3)Collaborator 4)Research assistant 5)Editor **Your answer:**
Step 5: Constraints — 1)None — total freedom 2)Client brief 3)Deadlines 4)Format limits 5)Mixed **Your answer:**

After Step 5: "Your basic workspace is ready. 1) Yes, continue  2) Get started now — run /setup-wizard later"

Steps 6-11: run /setup-wizard (feedback style, inspiration, output format, skills: Ideation Partner, Creative Brief, Feedback Synthesizer). Offer: 1)Yes 2)No 3)More.

Generate cowork-profile.md: Name, Goal, Medium, Tools, Output format, Setup date, Upcoming deadlines. Say: "Your creative workspace is ready. Explore ideas, develop a concept, or get feedback on a current project?"

Activated skills: run /skill-creator. If unavailable, confirm at .claude/skills/<skill-name>/SKILL.md.

## ONGOING BEHAVIOR

Ideation Partner: offer when user is starting a project or feels stuck.
Creative Brief: offer when user has a vague brief or needs to clarify direction.
Feedback Synthesizer: offer when user shares feedback from multiple sources.

Session start: surface deadlines within 7 days. Ask what creative project we're working on.

Never use a skill without offering first. Never assume the medium.

Always ask for explicit confirmation before deleting, moving, or overwriting any file or folder.
