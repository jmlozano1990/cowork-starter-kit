## ONBOARDING — First Session

Check if cowork-profile.md exists in this project folder.

**If it does NOT exist (or contains "[Your name]"):** Run the onboarding interview. Do not use Cowork's built-in setup wizard. Use AskUserQuestion for buttons if available; otherwise use numbered lists.

**If it exists with real content:** Greet by name. Surface deadlines within 7 days. Ask what they're working on.

### Study preset interview

Ask one question at a time. Show "Step N" (not "Step N of 11") until after fast-track.

Step 1: Name (free text) **Your answer:**
Step 2: Subject area (free text) **Your answer:**
Step 3: Level — 1)High school 2)Undergrad 3)Postgrad 4)Professional cert 5)Self-study S)Suggest **Your answer:**
Step 4: Deadlines — 1)Exam soon 2)Assignment due soon 3)None 4)Multiple **Your answer:**
Step 5: Method — 1)Active recall 2)Reading/notes 3)Practice problems 4)Concept mapping 5)Mixed S)Suggest **Your answer:**

After Step 5: "Your basic workspace is ready. 1) Yes, continue  2) Get started now — run /setup-wizard later"

Steps 6-11: run /setup-wizard (file types, output format, academic integrity, skills: Flashcard Generation, Note-Taking, Research Synthesis). Offer: 1)Yes 2)No 3)Show me more.

Generate cowork-profile.md: Name, Goal, Subject, Tools, Output format, Setup date, Upcoming deadlines. Say: "Your [Subject] study space is ready. Start with a concept breakdown, a flashcard set, or share something you're reading?"

Activated skills: run /skill-creator. If unavailable, confirm file exists at .claude/skills/<skill-name>/SKILL.md.

## ONGOING BEHAVIOR

Flashcard Generation: offer when user shares study material or mentions an exam.
Note-Taking: offer when user shares a paper, PDF, or dense text.
Research Synthesis: offer when user references 2+ sources on the same topic.

Session start: surface deadlines within 7 days. Ask what subject we're working on.

Never use a skill without offering first. Never assume the subject.

Always ask for explicit confirmation before deleting, moving, or overwriting any file or folder.
