## ONBOARDING — First Session

Check if cowork-profile.md exists in this project folder.

**If it exists with real content:** Greet by name. Surface deadlines within 7 days. Ask what to work on.

**If absent (or contains "[Your name]"):** Run onboarding. Use AskUserQuestion for buttons if available; otherwise use numbered lists.

### Phase 1 — Goal Discovery

Ask: "Welcome! What would you like to use this workspace for? Describe your goal — or type 'not sure' for suggestions."

**If uncertain** ("not sure", "maybe", "something like"):
"Three directions:
1. Study / Learning — exam prep, coursework, reading
2. Work / Projects — tracking, reports, stakeholder updates
3. Writing / Creating — articles, content, storytelling
Which fits? Or describe your goal."

**If goal matches a preset** (study, exam, research, writing, project, PM, creative, design, business, admin):
"That sounds like [Preset]. Want me to customize it?
1. Yes — use [Preset] as a scaffold  2. No — build from scratch"

**Novel goal:** "I'll build a [goal] workspace from scratch." Continue.

Hint: this starter is optimized for Business/Admin (email, reporting, scheduling, admin tasks).

### Phase 2 — Profile

- Step 1: Name (free text)
- Step 2: Role or context (free text)

### Phase 3 — Writing Profile

Say: "Communication style matters in business — these questions help me write in your voice, not like generic AI."

- Step 3: Tone — 1) Formal/corporate  2) Professional but warm  3) Direct and brief  4) Mixed
- Step 4: Audience — 1) Executives/leadership  2) Clients  3) Team  4) Mixed
- Step 5: Style — 1) Concise  2) Detailed  3) Structured/bullets  4) S) Suggest
- Step 6 (optional): "Paste a sentence from your writing to calibrate your voice — or 'skip'."
  Extract 2+ patterns if provided; state explicitly; do NOT store raw sample text.

Generate writing-profile.md: Tone & Voice, Style Preferences, Anti-AI Guidance, Workspace-Specific Rules, Pet Peeves.

**Fast-track:** "Basic workspace ready. 1) Continue — folder structure and skills  2) Get started now — run /setup-wizard later"

### Phase 4 — Full Setup

Run /setup-wizard for skills (Email Drafter, Document Summary, Action Items) and folder structure.

Generate cowork-profile.md: Name, Goal, Role, Tools, Setup date, Upcoming deadlines: (leave blank).

Activated skills: run /skill-creator. If unavailable, confirm file at .claude/skills/<name>/SKILL.md.
