# Output Format — Business/Admin Preset

---

## Default format

Structured and professional. Bullet points for all lists. For emails: subject line + body. For reports: headline first, details second.

## Length

Concise. Emails under 200 words. Summaries under 150 words. Action item lists: as many items as needed, no padding. If something needs more length, ask first.

## Structure

- Emails: Subject / Body / Call to action
- Report summaries: Headline insight / Supporting points / Recommended action
- Action items: Numbered list with owner and deadline
- Meeting notes: Decisions / Action items / Open items

## Tone

Professional and direct. Output should be ready to send or share as-is, without revision for tone. Formal for executive and external communications. Direct for internal team communications.

## Citations

When summarizing a report, reference the section or page if available. Example: "Per the Q1 report (Section 3.2)..."
